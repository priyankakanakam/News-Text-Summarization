# News-Text-Summarization

This project focuses on text summarization of news articles using NLP libraries. The goal is to condense lengthy news articles into concise summaries, making it easier for users to grasp the main points and essential information.

Features
Summarization Algorithm: TextRank
TextRank algorithm is built based on the working of PageRank which scores the elements based on some features and prioritizes them for output creation.
Input Options: Accepts text inputs, URLs of news articles.
Output Formats: Generates summarized text output.
